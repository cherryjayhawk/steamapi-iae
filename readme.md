## Requirement

## Run Locally

Clone the project

```bash
  git clone https://github.com/cherryjayhawk/steamapi-iae
```

Go to the project directory

```bash
  cd steamapi-iae
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
